TEXTO DE HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. HTML utiliza una serie de elementos o etiquetas para estructurar el contenido en la web.

Etiquetas Comunes

Algunas etiquetas HTML comunes incluyen:

  • <p>: Define un párrafo.

  • <a>: Define un enlace.
    •

  • <h1> a <h6>: Define encabezados de diferentes niveles.

    • Ejemplos de Etiquetas

    Encabezados

    Hay seis niveles de encabezados en HTML:

    <h1>Este es un encabezado de nivel 1</h1>

    <h2>Este es un encabezado de nivel 2</h2>

    <h3>Este es un encabezado de nivel 3</h3>

    Párrafos

    <p>Este es un párrafo.</p>

    Enlaces

    <a href="https://www.ejemplo.com">Este es un enlace</a>

    Negritas

    <strong">Este es un enlace</strong>

    Cursiva

    <cursiva">Este es un enlace</cursiva>
