Ejemplos de HTML

Etiquetas Básicas de HTML

Estas son algunas de las etiquetas básicas de HTML con ejemplos:

<html> y <body>

<html>
  <head></head>
  <body>
    <h1>Título</h1>
    <p>Un párrafo de texto.</p>
  </body>
</html>

<a> (Enlace)

<a href="https://www.example.com">Este es un enlace</a>

<img> (Imagen)

<img src="imagen.jpg" alt="Descripción de la imagen">

<table> (Tabla)

<table>
  <tr>
    <th>Encabezado</th>
    <th>Encabezado</th>
  </tr>
  <tr>
    <td>Celda</td>
    <td>Celda</td>
  </tr>
</table>

Ejemplo de Frames

Los marcos (frameset) definen la organización de los marcos dentro de la ventana del usuario. Este ejemplo es para fines educativos, ya que el uso de frames está obsoleto en HTML5.

Ejemplo de Código para Frames

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN"
    "http://www.w3.org/TR/html4/frameset.dtd">
<html>
<head>
  <title>Un documento simple con marcos</title>
</head>
<frameset rows="20%,80%">
  <frameset cols="20%, 80%">
    <frame src="http://developer.mozilla.org/wiki-images/firefoxlogo.png">
    <frame src="http://html.conclase.net/w3c/html401.dtd.html#frame">
  </frameset>
  <frame src="http://html.conclase.net/w3c/html401.dtd.html#edef-FRAME">
</frameset>
</html>
